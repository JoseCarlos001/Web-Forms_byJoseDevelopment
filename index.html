Definición
HTML es el que permite que un navegador sepa lo que esta leyendo
y como debe mostrarlo al usuario. De esta forma, los navegadores
saben donde deben mostrar el texto, donde va una imagen o donde hay un enlace.
Aplicaciones
Las aplicaciones HTML5 son aplicaciones web y deben ejecutarse
utilizando el navegador del sistema operativo subyacente.
